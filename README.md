# clothing
Power BI project on mens clothing. The project aims to show top 5 categories for various metrics including, top 5 brands by average discount percentage, top 5 brands by highest average profit percentage, top 5 brands by highest number of varieties, top 5 brands by highest average sales price, and bottom 5 brands by average profit percentage.

The project highlights skills in data cleaning, and visualization by creating a donut chart, area chart, circle graph, bar chart, and ribbon chart.
